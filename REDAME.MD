Agrobanck - Lead Capture System
Este é um sistema de captura de leads de alta conversão desenvolvido para a Agrobanck. O projeto foca em uma experiência de usuário (UX) fluida, com validações em tempo real e automação de formatação para garantir que os dados cheguem organizados via WhatsApp.

✨ Funcionalidades
Animação de Texto Dinâmica: Frases que alternam no cabeçalho para destacar os benefícios do serviço.

Barra de Progresso: Indicador visual que aumenta à medida que o usuário preenche os campos obrigatórios.

Padronização Automática:

Capitalização: Nomes, Cidades e Negócios têm a primeira letra de cada palavra convertida para maiúscula automaticamente.

Localidade: Formata automaticamente para o padrão Cidade/UF (ex: Cuiabá/MT).

WhatsApp: Máscara inteligente que aceita 11 dígitos e permite apagar todo o campo sem travar.

Sistema de Travas (Segurança):

Validação real de e-mail (exige @ e .).

Obrigatoriedade de preenchimento do campo "Outro" quando selecionado no perfil.

Bloqueio de envio para dados incompletos ou mal formatados.

🛠️ Tecnologias Utilizadas
HTML5 / CSS3: Estrutura e estilização moderna com foco no setor agro.

JavaScript (Vanilla): Lógica de validação, máscaras e integração com a API do WhatsApp.

Vercel: Hospedagem e deploy contínuo integrado ao GitHub.
